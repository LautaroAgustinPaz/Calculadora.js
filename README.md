# 🧮 Calculadora en JavaScript

Proyecto de práctica donde desarrollé una calculadora funcional usando HTML, CSS y JavaScript.

## 🚀 Demo

👉 [🚀 Ver Demo en vivo](calculadora-aquamarine.netlify.app)

## 🛠️ Tecnologías
- HTML5
- CSS3 (Grid)
- JavaScript

## ⚙️ Funcionalidades
- Operaciones básicas (+, -, *, /)
- Botón de borrado (←)
- Reset (C)
- Manejo de errores
- Interfaz responsive básica

## 🧠 Lo que aprendí
- Manipulación del DOM
- Manejo de eventos
- Lógica de calculadora
- Debugging y resolución de errores
